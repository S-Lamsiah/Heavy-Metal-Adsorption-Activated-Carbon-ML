 ## **Description:**
 This repository contains the Python notebook and database used to predict the adsorption capacity of heavy metals by activated carbon using machine learning models. This notebook implements the method described in the associated scientific article.
 
 ## **Dataset:**
 The database used in this study contains 1,528 observations and 21 variables, drawn from various experimental studies. It includes: 
 1) Properties of heavy metals (molar mass, van der Waals radius, hydrated radius, electronegativity)
 2) Textural properties of activated carbon (BET surface area, pore volume, pore diameter)
 3) Experimental conditions (pH, temperature, dose, contact time, initial concentration)  4)Target variable: adsorption capacity (mg/g).
